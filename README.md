# Prayagraj Tourism Website

## About Project
This is a travel blog website that provides information about famous tourist places, cultural heritage, and attractions of Prayagraj. 
The website helps users explore the history, temples, and popular destinations of Prayagraj.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Information about famous places in Prayagraj
- Image gallery
- Attractive user interface
- Responsive design

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `Travell-blog.html` in your browser.
